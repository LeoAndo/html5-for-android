段組のサンプル

これは段組のサンプルです。CSS3からは手軽に段組ができるようになりました。これまでのようにfloatなどを駆使する必要はありません。

この段組ですが、実はWeb普及期に登場したブラウザであるNetscape 3にはmulticolタグがあり、これを利用すると簡単に段組を行うことができました。ただ、その後急速に利用されるようになったInternet Explorerではサポートされず、お蔵入りになってしまったような歴史があります。

CSS3の段組のプロパティを見ると基本的にはmulticolタグと同じです。というより、段組を行う場合にはどれも似たような感じになってしまうでしょう。
